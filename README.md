# Python Data Cleaning Project.
A school project, using a very dirty dataset, focused on data exploration, data cleaning and data visualization using libraries such as NumPy, Pandas, Matplotlib, etc., to handle missing values, removing duplicates, correcting data types.

Objective:
- Derive insights from the dataset.

Tools used:
- Pandas.
- NumPy.
- Matplotlib.

Key Insights:
- Laptops generated more revenue than other products.
- Overall customer rating was good. Need toc be better.
- Proper standardization need to be prioritized.
